# Лабораторна робота №2: Використання UART для виведення інформації
## Курс: # Програмування мікроконтролерів (ПМк) | УжНУ

Цей репозиторій містить проєкт для середовища **PSoC Creator 4.4**, спрямований на вивчення роботи інтерфейсу UART. Програма передає текстові повідомлення про стан кнопок та світлодіодів з плати PSoC 4 Pioneer Kit на комп'ютер.

---

## 📋 Суть роботи

- Налаштування компонента **Software Transmit UART** (115200 бод)
- Керування RGB-світлодіодами
- Зчитування стану кнопки
- Виведення повідомлень у термінал (`Button pressed`, `Button released`)

---

## 🛠 Необхідне ПЗ

- **PSoC Creator 4.4**
- **PSoC Programmer**
- Термінал:
  - **PuTTY**
  - **TeraTerm**

---

## 🚀 Запуск

### Клонування
```bash
git clone https://github.com/ERRORES241/PMk_Lab2_UzhNU.git
````

### Відкриття

Відкрити `.cywrk` у **PSoC Creator 4.4**

### Оновлення компонентів

Погодитися на **Update Components**, якщо запропоновано

### Налаштування плати

- SW4.1 — OFF
    
- SW4.2 — OFF
    
- SW4.3 — ON
    

### Збірка і прошивка

- `Shift + F6` — Build
    
- `Ctrl + F5` — Program
    

### Термінал

- COM-порт: KitProg USB-UART
    
- Швидкість: **115200**
    

Після Reset з’являються повідомлення у терміналі
